Leaflet Samples

Leaflet is a JavaScript library for creating interactive maps

Utility Outage Clusters

Utility Outage Clusters is a Leaflet map showcasing marker clusters. As you zoom out, circle markers that represent individual outages converge into clusters. Outage data was fabricated for twenty outages and represent locations around Washington DC.
