This site is compilation of resources for open-source BlueGranite projects.

flexdashboard

flexdashboards allow users to create R dashboards using Markdown. Our example, Fiji Earthquakes is a sample R flexdashboard showing data from the R quakes dataset. It uses Leaflet, ThreeJS, and ggvis.

  • Learn more

leaflet

Leaflet is a JavaScript library for creating interactive maps. Our example, Utility Outage Clusters is a Leaflet map showcasing marker clusters. As you zoom out, circle markers that represent individual outages converge into clusters. Outage data was fabricated for twenty outages and represent locations around Washington DC.

  • Learn more

Hexbin Scatterplot for Power BI

The Hexbin Scatterplot is a custom visual for Microsoft Power BI that displays points on top of hexagonal "bins". Color saturation for the hexbins shows the density of points within each bin, with darker bins showing more points.

  • Learn more

rDataTable

Based on the DT package, rDataTable is an interactive Power BI R custom visual that helps you easily explore table data. It is pending approval in the Office Store / AppSource.

  • Learn more

Image Timeline for Power BI

The Image Timeline is a custom visual for Microsoft Power BI that displays events along a horizontal timeline. Events appear as either circles or custom images.

  • Learn more
